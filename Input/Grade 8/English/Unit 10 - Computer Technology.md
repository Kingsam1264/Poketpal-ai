### Machines with Memories  

Computers and calculators are a revolutionary development in the history of technology. They are fundamentally different from all other machines because they have a memory. This memory stores instructions and information.  

In a calculator, the instructions are the various methods of arithmetic. These are permanently remembered by the machine and cannot be altered or added to. The information consists of the numbers keyed in.  

A calculator requires:  
- An **input unit** to feed in numbers  
- A **processing unit** to make the calculation  
- A **memory unit**  
- An **output unit** to display the result  

The memory unit stores the arithmetic instructions and holds temporary results during calculation.  

The input unit has keys for numbers and operations. Pressing a key closes contacts and sends a signal to the processing unit, where the binary code for that key is stored in memory. The processing unit also sends the code to the display.  

**Fig: Parts of a computer or a calculator**  
Input → Processing → Output  
↑  
Memory Unit  

A computer contains the same basic four elements as a calculator. It differs in that its memory can be given a different set of instructions called a **computer program** for different tasks. A program can turn a computer into, for example, a game player, a word processor, a paint box, or a musical instrument.  

Computer programs consist of long sequences of simple instructions. A computer knows how to multiply because you tell it what to do — by adding the first number to itself the second number of times. Computers can perform millions of instructions in a matter of seconds.

### Lesson 11: Adjectives and Adverbs  

**Examples:**  
i. We had a get-together party. Though it was beautiful, it was too short; the time passed **very quickly**.  
ii. There was a car accident yesterday. Two people were **seriously injured** in the accident.  

**Activity:**  
1. How are the words in bold formed?  
2. What is their function?  
3. How are they used?  

**Note:**  
- **Quickly** and **seriously** are **adverbs**.  
- Many adverbs are formed by adding **-ly** to adjectives.  

| Adjective | Adverb     |
|-----------|------------|
| quick     | quickly    |
| serious   | seriously  |
| careful   | carefully  |
| bad       | badly      |
| heavy     | heavily    |
| terrible  | terribly   |

**Not all -ly words are adverbs.** Some adjectives also end in **-ly**, e.g., **friendly**, **lovely**, **elderly**, **lonely**.  

**Usage:**  
- **Adjectives** describe nouns:  
  - *Mamo is a **careful** driver.*  
  - *We didn’t go out because of the **heavy** rain.*  

- **Adverbs** describe verbs, adjectives, or other adverbs:  
  - *Mamo drove **carefully** along the narrow road.*  
  - *It was raining **heavily**.*  

**Examples:**  
- *It is a **reasonably cheap** restaurant, and the food is **extremely good**.*  
- *Hawa learns languages **incredibly quickly**.*  
- *Two people were **seriously injured** in the accident.*  

**Special Cases:**  
- **Fast**, **late**, **hard**, **early** are both adjectives and adverbs.  
- **Hardly** has a negative meaning: *We hardly know each other.*  
- **Good** (adjective) vs. **Well** (adverb or adjective meaning healthy):  
  - *Your English is very **good**. You speak it very **well**.*  
  - *I am very **well**, thank you.*

## APPENDIX  

### Regular and Irregular Verbs  

**Regular Verbs:** Add **-ed** or **-d** to form past tense:  
- like → liked  
- carry → carried  
- drop → dropped  
- remember → remembered  

**Irregular Verbs:** Past tense is not formed by regular rules.  

**Examples:**  
| Base Form | Simple Past | Past Participle |
|-----------|-------------|------------------|
| go        | went        | gone             |
| eat       | ate         | eaten            |
| see       | saw         | seen             |
| write     | wrote       | written          |
| speak     | spoke       | spoken           |
