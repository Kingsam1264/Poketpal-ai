**Definition 3.2:** A constant (a number), a variable, product or quotient of a number and variable is called a term.

**Example:** 4 is a term. xy is a term. 12xy - 4 is not a term. 3x/y is a term. x - y is not a term.

**Definition 3.3:** In the product of a number and variable, the numerical factor of the term is called numerical coefficient.

**Example 2:**
a. The numerical coefficient of 6ab is 6.
b. The numerical coefficient of -3xy is -3
c. The numerical coefficient of x is 1.
d. The numerical coefficient of -2ab/3 is -2/3.

**Definition 3.4:** Like terms are terms whose variable and exponent of variable are exactly the same but they may differ in their numerical coefficients. Terms which are not like terms are called unlike terms. A term which does not contain a variable is called constant term. All constant terms are like terms.

**Example 3:**
- 2 and -5 are like terms, because they are constant terms.
- 3a²b² and -2/3b²a² are like terms.
- -20abc and acb are like terms.
- 3xyz and xy²z are unlike terms.
- -2a³b³c³ and -2a³b²c³ are unlike terms.
- 3xy and 3yx are like terms.

**Definition 3.5:** Algebraic expressions are formed by using numbers, letters and the operation of addition, subtraction, multiplication and division.

**Example 4:** 12, 2x - 3, a + 10, 3x, 3/4a, 3x, 3x + 2a - 5, etc. are algebraic expressions.

**Definition 3.6:** An algebraic expression in algebra which contains one term is called a monomial.

**Example:** 2ab is a monomial.

**Definition 3.7:** An algebraic expression in algebra which contains two terms is called a binomial.

**Example:** 2x + 3y - 3, 12 is binomial.

**Definition 3.8:** An algebraic expression in algebra which contains three terms is called a trinomial.

**Example:** x - y - z, 2x + 3y - 3, 12 are trinomials.

**Simplifying Algebraic Expressions**

Basic steps:
1. Remove brackets using distributive property.
2. Collect like terms and add or subtract like terms.

**Note:**
1. When you add or subtract like terms, add or subtract their numerical coefficients. Example: -3x + 5x = (-3 + 5)x = 2x
2. Unlike terms cannot be added or subtracted. Example: 5x + 2y cannot be added.
3. When you remove brackets using distributive property be careful of signs.
4. When you multiply a number with variable, you can write the number as coefficient without multiplication sign.

**Definition 3.9:** Two different algebraic expressions connected by equal (=) sign is called equation.

**Definition 3.10:** A Linear equation in one variable x is an equation which can be written in standard form ax + b = 0, where a and b are constant numbers with a ≠ 0.

**Note:** An equation of a single variable in which the highest exponent of the variable is one is a linear equation.

**Example 1:** Which are linear equations?
a. 2x = 2
c. 2x² + 3 = 2x - 1
b. x + 3 = 2x - 1
d. x = 2x³ - 6

**Solution:** a and b are linear equations, because the highest exponent of variable is one. c and d are not linear equations, because the highest exponent of the variable is not one.

**Note:** All equations have two sides with respect to the equal sign: Left hand side (L.H.S) and right-hand side (R.H.S)

**Rules of transforming equations to simpler form:**

**Fig 3.1: Balance Scale Demonstrating Equation Rules**
The figure shows a simple balance scale demonstrating the four transformation rules:
- Adding the same number to both sides maintains balance
- Subtracting the same number from both sides maintains balance
- Multiplying both sides by the same non-zero number maintains balance
- Dividing both sides by the same non-zero number maintains balance

Rule 1: Adding the same number to both sides of equation will not affect the equation.
Rule 2: Subtracting the same number from both sides of equation will not affect the equation.
Rule 3: Multiplying both sides of equation with the same non-zero number will not affect the equation.
Rule 4: Dividing both sides of equation with the same non-zero number will not affect the equation.

**Example 2:** Solve equations:
a. x - 10 = 6

**Solution:**
a. x - 10 = 6 → x = 16
b. 10 - x = 6 → x = 4
c. x + 4 = -6 → x = -10

**Example 3:** Solve equations:

**Definition 3.12:** The two perpendicularly intersecting horizontal and vertical number lines together set up a plane is called Cartesian coordinate plane.

**Locating points on Cartesian Plane**

Points on Cartesian plane are described by two numbers (a, b) that are called coordinates. The first number a is the horizontal position from the origin (x-coordinate or abscissa). The second number b is the vertical position from the origin (y-coordinate or ordinate).

**Steps to locate point (a, b):**
1. Move a units right if a is positive, left if a is negative
2. Move b units up if b is positive, down if b is negative

**Example 10:** Locate points:
a. (3, 2)
b. (-3, 2)
c. (2, 0)
d. (0, -4)

**Solution:**
a. Move 3 units right, then 2 units up
b. Move 3 units left, then 2 units up
c. Move 2 units right, no vertical movement
d. No horizontal movement, move 4 units down

**Note:** 
- Any point (a, 0) lies on x-axis
- Any point (0, b) lies on y-axis

**Determining coordinates of a point:**

**Fig 3.2: Determining Coordinates of a Point**
The figure shows a point P on the Cartesian plane. A vertical line through P crosses the x-axis at a, and a horizontal line through P crosses the y-axis at b. Therefore, the coordinates of P are (a, b).

Example: For point A, the vertical line crosses x-axis at 3 and horizontal line crosses y-axis at 4, so A is (3, 4).

## Summary for Unit 3

- **Variable:** Any letter or symbol representing unknown value
- **Term:** Constant, variable, product or quotient of number and variable
- **Numerical coefficient:** Numerical factor in product
- **Like terms:** Same variable and exponent (may differ in coefficient)
- **Algebraic expression:** Formed by numbers, letters, +, -, ×, ÷
- **Equation:** Two expressions connected by =
- **Linear equation:** Highest exponent of variable is 1 (ax + b = 0, a ≠ 0)
- **Transformation rules:** Add/subtract/multiply/divide both sides by same non-zero number
- **Bracket properties:** Distributive property
- **Cartesian coordinates:** (x, y) ordered pairs
- **Quadrants:** 4 regions divided by axes
- **Graphs:** x = a (vertical line), y = b (horizontal line), y = mx (line through origin)
- **Word problems:** Read, define variable, write equation, solve, interpret, check
