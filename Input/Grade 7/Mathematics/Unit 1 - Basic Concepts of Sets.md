### Definition of a Set
A set is a collection of well-defined objects. The individual objects in a set are called members or elements of the set. The braces "{" and "}" are used to enclose the members of the set. Each element of a set separated by comma are listed between a pair of braces {} without repetition and without concern about the order of their appearance. We use capital letters like A, B, C, etc. to denote sets. Small case letters are used to denote elements of the set. For example, the set of all vowels in English alphabet can be represented by a single letter A and described as:

**A = {a, e, i, o, u}**

If a is a member of set A, then we write **a ∈ A** (read as "a is an element of set A"). If a is not a member of set A, then we write **a ∉ A** (read as "a is not an element of set A").

### Example 1:
If N = {2, 4, 6, 8, 10}, then which of the following are true and which are false?

a. 6 ∈ N  
b. 12 ∉ N  
c. 3 ∈ N  
d. 2 ∉ N

**Solution:**
a. **True**, because 6 is found in the set N.  
b. **True**, because 12 is not found in set N.  
c. **False**, because 3 is not an element of set N.  
d. **False**, because 2 is an element of set N.

### Example 2:

b. The collection of five most talented writers of Ethiopia is not a set, since no rule has been given for deciding whether a given writer is talented or not.

## 1.2 Relations among Sets
By the end of this section you will be able to:
- Describe relationship among sets such as proper subset, subset, equal and equivalent sets.

### A. Equal Sets
**Definition 1.2:** Two non-empty sets A and B are said to be equal, if they have exactly the same elements and write, **A = B**.

#### Example 1:
Let A = {p, q, r, s} and B = {q, r, p, s}. Are A and B equal?

**Solution:** A = B (since they contain the same elements, order doesn't matter)

#### Example 2:
Let C = {1, 1, 2, 2, 3} and D = {1, 2, 3}. Are C and D equal?

**Solution:** C = D (since repeated elements don't change the set)

#### Example 3:
Let E = {u, v, x} and F = {x, u, 3}. Are E and F equal?

**Solution:** Elements in set E are not exactly the same as the elements in set F, so E and F are not equal (i.e., E ≠ F).

### B. Equivalent Sets
**Definition 1.3:** Two sets A and B are said to be equivalent, written **A ↔ B**, if and only if they have equal number of elements.

#### Example 4:
Let E = {u, v, x} and F = {x, u, 3}. Is E equivalent to F?

**Solution:** E ↔ F (both have 3 elements)

#### Example 5:
Let A = {p, q, r, s} and B = {q, r, p}. Are A and B equivalent?

**Solution:** Set A and B are not equivalent, since the number of elements in A and B are not equal.

**Note:** All equal sets are equivalent, but all equivalent sets are not equal sets.

#### Example 6:
Let C = {1, 2, 3} and D = {3, 2, 1}. Are C and D equivalent?

**Solution:** Yes, C ↔ D (both have 3 elements, and they are also equal).

### C. Subsets
**Definition 1.4:** A set A is said to be a subset of set B if every element of A is also an element of B, and write, **A ⊆ B**.

#### Example 7:
Let A = {2, 3, 5} and B = {2, 3, 5, 7, 9}. Is A a subset of B?

**Solution:** All elements in A are also in B, so A ⊆ B.

#### Example 8:
Let B = {a, e, i, o, u} and A = {a, c, e}. Is A a subset of B?

**Solution:** There is an element in A which is not in B, for example c ∈ A, but c ∉ B. Therefore A is not subset of B (i.e., A ⊄ B).

#### Example 9:
Let M = { } and N = {a, d}, then which of the following is true?
a. M ⊆ N
b. N ⊆ N

**Solution:**
a. { } contains no element at all, so there is no element of { } which is not contained in N, hence M ⊆ N is true.
b. All elements in N are also in N, hence N ⊆ N.

**Note:**
1. Empty set is a subset of any set (i.e., for any set A, ∅ ⊆ A).
2. Any set is a subset of itself (i.e., for any set A, A ⊆ A).

#### Example 10:
Let A = {1, 2}, then list all subsets of set A.

**Solution:** {1} ⊆ A, {2} ⊆ A, {1, 2} ⊆ A and ∅ ⊆ A. Therefore all subsets of set A are {1}, {2}, {1, 2} and ∅.

### D. Proper Subsets
**Definition 1.5:** A set A is said to be a proper subset of set B, written **A ⊂ B**, if and only if A is a subset of B and B has at least one more element.

**Note:**
1. Empty set is a proper subset of any other set (i.e., for any set A different from empty set, ∅ ⊂ A).
2. Any set is not a proper subset of itself (i.e., for any set A, A ⊄ A).

#### Example 11:
Let A = {a, e, c} and B = {a, c, d, e}. Is A a proper subset of B?

**Solution:** All elements in A are also in B, and B has additional element(s), so A ⊂ B.

#### Example 12:
Let D = ∅ and E = {a, d}, then which of the following is true?
a. D ⊂ E
b. E ⊂ E

**Solution:**
a. Since empty set is a proper subset of any other set, D ⊂ E is true.
b. Since any set is not a proper subset of itself, E ⊂ E is false.

#### Example 13:
Let A = {a, c}. List all proper subsets of set A.

**Solution:** {a} ⊂ A, {c} ⊂ A, and ∅ ⊂ A. Therefore all proper subsets of set A are {a}, {c}, and ∅.

## 1.3 Operations on Sets
By the end of this section you will be able to:
- Determine the intersection of two given sets.
- Determine the union of two given sets.
- Use Venn diagram to represent union and intersection of two sets.

### 1.3.1 The Union of Sets
**Definition 1.6:** The union of two sets A and B, denoted by **A ∪ B** and read "A union B", is the set of all elements which belong to A or B or both.

**Fig 5: Venn Diagram for Union of Sets**  
*The image shows two overlapping circles labeled A and B. The entire area of both circles (including the overlapping region) is shaded, representing A ∪ B.*

#### Example 1:
If A = {3, 4, 5, 6} and B = {4, 6, 8, 10}, then find A ∪ B.

**Solution:** A ∪ B = {3, 4, 5, 6, 8, 10}

#### Example 2:
If A = {a, e, i} and B = {a, e, i, o}, then find A ∪ B.

**Solution:** A ∪ B = {a, e, i, o}

### 1.3.2 The Intersection of Sets
**Definition 1.7:** The intersection of two sets A and B, denoted by **A ∩ B** and read "A intersection B", is the set of all elements which belong to both A and B.

**Fig 6: Venn Diagram for Intersection of Sets**  
*The image shows two overlapping circles labeled A and B. Only the overlapping region (where the circles intersect) is shaded, representing A ∩ B.*

#### Example 3:
If A = {3, 4, 5, 6} and B = {4, 6, 8, 10}, then find A ∩ B.

**Solution:** A ∩ B = {4, 6}

#### Example 4:
If A = {a, e, i, o} and B = {a, e, i}, then find A ∩ B.

**Solution:** A ∩ B = {a, e, i}

#### Example 5:
If A = {4, 7, 9} and B = {a, e, i}, then find A ∩ B.

**Solution:** A ∩ B = ∅ (empty set)

## Summary for Unit One

- **Set** is a collection of well-defined objects.
- The individual objects in a set are called **members or elements** of the set.
- A set which does not have an element is called **empty set**.
- Two sets A and B are said to be **equal**, if they have exactly the same elements.
- A set A is said to be a **subset** of set B, written A ⊆ B if every element of A is also an element of B.
- A set A is said to be a **proper subset** of set B, written A ⊂ B, if and only if A is a subset of B and B has at least one more element.
- The **union** of two sets A and B, denoted by A ∪ B, is the set of all elements which belong to A or B or both.
- The **intersection** of two sets A and B, denoted by A ∩ B, is the set of all elements which belong to both A and B.
